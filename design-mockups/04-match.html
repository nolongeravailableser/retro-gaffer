What changed vs the current MatchView

  • Decisions are a true overlay, not a card in the scroll. Half-time talks and injury subs currently render inline in the scrollable feed — if you've scrolled, the match pauses with the decision below the fold and the game appears frozen. The overlay dims the pitch, blocks scrolling, and each option carries its effect inline (“+ATK / −DEF until FT”), so the choice is informed rather than vibes. Alternative considered: auto-scroll the feed to the decision card. Rejected — fights the user's scroll position and still competes with the live ratings panel; a modal moment is honest about the fact the game is paused.
  • One screen, three zones, no collapsibles mid-match. Scoreboard (kits, score, minute, thin progress) / pitch / 3-line caption feed. The collapsible live-ratings panel goes — live ratings move to full-time, where they're actually read. Speed controls remain visible while paused (greyed), fixing “can't change speed at a decision point”.
  • Full-time is a receipt, not a sprawl. The single most important post-match question in an economy game is “what did this match do to my money?” — today the net payout is one long sentence at the bottom of a scroll. The receipt itemises Result / Gate / Interest / Streak / Wages / Upkeep → Net, ending in the new balance. Team news (bans/injuries) sits directly under the banner with the consequence spelled out (“you'll need a MID for MW8”) and a one-tap “Fix XI” route. Alternative: keep the prose sentence (more “retro ticker” in tone). Rejected — the prose hides signed numbers; a tabular receipt is still period-correct for a CM homage (it's a teletext payslip).
  • Report stays. Timeline + ratings grid survive below the receipt — they're good. MOTM gets a label (“MOTM”) instead of a bare star.
